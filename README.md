# 2026 Westtown Robotics Team - Metal Moose

Welcome to the Metal Moose codebase for the 2026 season.

## Overview

This repository contains the robot code for Westtown School's FRC robotics team, Metal Moose.

## DevOps Stack

- **WPILib**: Robot programming framework for FIRST Robotics Competition
- **Gradle**: Build automation and dependency management
- **JUnit 6**: Unit testing framework

## Getting Started

1. Clone this repository
2. Ensure you have the WPILib development environment installed
3. Run `./gradlew build` to compile the project
4. Run `./gradlew test` to execute unit tests

## Project Structure

```
src/
├── main/java/    # Main robot code
└── test/java/    # Test code
```

## Contributing

Follow team coding standards and ensure all tests pass before submitting changes.
